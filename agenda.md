# Lagebild KI - 2h Session

> Praxis-Session für Kollegen und Freunde: Wir bauen live einen "Jira Copilot" – von der Idee bis zum Deployment, komplett mit KI.

## Durchlauf-Projekt: Jira Copilot

Ein VS Code Projekt das Jira-Daten intelligent aufbereitet: Daily-Vorschläge, Sprint-Planung, Kapazität, Abrechnung – gebaut live im Editor.

## Zeitplan (120 Min.)

| Zeit       | Modul | Thema                                          | Dauer  |
|------------|-------|-------------------------------------------------|--------|
| 00:00      | Intro | Über mich, Lagebild KI, Projekt-Vorstellung     | 10 min |
| 00:10      | 1     | Requirements – KI schreibt die Spec              | 20 min |
| 00:30      | 2     | Anwendung – KI baut das Projekt in VS Code       | 25 min |
| 00:55      |       | **Pause**                                        | 5 min  |
| 01:00      | 3     | Dokumentation – KI generiert Docs aus Code       | 20 min |
| 01:20      | 4     | Infrastruktur – CI/CD, Container, k3s Deployment | 20 min |
| 01:40      | 5     | Agent als PM – Agent macht alles allein           | 15 min |
| 01:55      | Outro | Q&A & Diskussion                                 | 5 min  |

## Roter Faden

Ein Projekt, fünf Perspektiven:
1. **Requirements** – "Was soll es können?" → Strukturierte Spec
2. **Anwendung** – "Bau es." → Lauffähiger Code mit Tests
3. **Dokumentation** – "Erklär es." → README, Setup, Architecture
4. **Infrastruktur** – "Bring es live." → CI/CD, Container, k3s
5. **Agent als PM** – "Mach das nächste Feature allein." → End-to-End

## Zielgruppe

- Engineering Manager, Project Manager, Team Leads
- Keine KI-Vorkenntnisse nötig
- Grundverständnis von Softwareentwicklung / IT hilfreich
