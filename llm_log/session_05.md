### 2. **GitHub Actions Cron — The Quick Win**

A `.github/workflows/jira-lagebild.yml` could:

```yaml
on:
  schedule:
    - cron: '0 * * * *'  # every hour
  workflow_dispatch:       # manual trigger for demos

jobs:
  lagebild:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - run: npm ci
      - run: npx tsx scripts/jira-report.ts
        env:
          JIRA_API_TOKEN: ${{ secrets.JIRA_API_TOKEN }}
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
      - run: # post to Slack / send email / create Jira comment
```

The `jira-report.ts` script would:
- Fetch open tickets, sprint progress, recent changes from Jira API
- Call Claude API to analyze diffs, summarize, and flag issues
- Emit alerts via Slack webhook or email if anomalies are found (e.g., blocked tickets > 2 days, sprint burndown off track)
