# Modul 5: Coding Agent als Projekt Manager (15 Min.)

> Der Agent bekommt ein Feature und macht alles – Requirements bis Deployment

## Einstieg (für Teilnehmer)

"Jetzt kommt der Moment der Wahrheit. Wir geben dem Agent EIN Feature und schauen zu, wie er alles macht was wir in Modul 1-4 einzeln gezeigt haben."

## Live-Demo

1. **Feature-Request:** "Füge Kapazitätenplanung hinzu – wer ist verfügbar, wer überlastet?"
2. **Agent plant:** Zerlegt in Tasks, schätzt Scope
3. **Agent baut:** Code, Tests, Doku, Infra-Updates
4. **Agent liefert:** PR mit allem drin

## Talking Points

- [ ] Der Agent hat Kontext über das gesamte Projekt
- [ ] Eigene Regeln und Team-Daten beeinflussen das Ergebnis
- [ ] PM entscheidet, Agent führt aus
- [ ] Grenzen: Wo braucht es weiterhin den Menschen?

## Takeaway

> "Der Agent ist nicht der Ersatz für den PM – er ist das mächtigste Werkzeug im Werkzeugkasten."
